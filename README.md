# 旅行典藏 PWA

可擴充的旅行典藏模板系統。第一個模板為 `Passport 1.0`，輸出 9:16 Instagram 限時動態，並支援直式照片與完整 16:9 橫式照片。

第一版核心功能已完成，進度與後續驗證項目請參閱 [HANDOFF.md](./HANDOFF.md)。

## 已完成

- 中文操作、英文成品輸出。
- Passport 1.0 直式／完整橫式照片模式。
- 旅行資料保存、沿用上一張與自動流水號。
- 本機照片保存、Instagram及高解析典藏版匯出。
- 可安裝與離線啟動的PWA。

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```
