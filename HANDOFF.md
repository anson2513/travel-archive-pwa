# 旅行典藏 PWA｜開發交接

更新日期：2026-06-30

## 目前完成

- React、TypeScript、Vite 與可安裝 PWA。
- 可擴充模板登錄架構；旅行資料與模板彼此分離。
- Passport 1.0 固定輸出 1080 × 1920，另有 2160 × 3840 高解析典藏版。
- 直式照片模式與完整保留 16:9 橫式照片模式。
- 深色底材、深藍票券、實際挖空的左右半圓票根與撕線。
- 全中文操作介面，英文成品輸出欄位。
- 18種中文天氣選項，只需手動輸入溫度。
- Trip Profile 旅行資料保存與套用。
- 「新增下一張並沿用資料」：保留上一張所有輸入，照片、流水號與 Barcode 重新產生。
- IndexedDB 原始照片保存、localStorage 典藏資料保存與自動儲存。
- Instagram 安全區預覽、照片解析度警告與JPEG品質設定。
- 桌面及手機響應式介面。
- `npm run build` 建置通過，PWA Service Worker 已產生。

## 已驗證流程

- 新增典藏。
- 中英文城市、國家、地點輸入。
- 天氣選擇、氣溫及相機輸入。
- 航線、航班、座位與旅行資料保存。
- 旅行短句。
- 新增下一張並保留上一張資料，流水號由001增加為002。
- 手機版典藏庫響應式版面。
